### Расписание.

Расписание имеет 3 вхождения по 4 периода.

	1. С понельника по пятницу.
	2. Суббота.
	3. Воскресенье.

Один период - часы, минуты и температура включения.
 
У этого термостата есть `Эко` режим. При его включении, установочная температура считывается не из стандартной настройки `Occupied heating setpoint`, а из `Эко-температуры`. Которая есть в меню самого термостата под номером `b`. Но наружу эта темперература термостатом не отдается и из-вне не принимается, поэтому управлять ей удаленно нельзя. Т.е. по сути функция `Эко` минимум неудобна, а максимум бесполезна.
